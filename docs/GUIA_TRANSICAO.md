# 📊 Guia de Transição: Excel → OKR Dashboard

Este guia ajuda você a encontrar na nova plataforma cada elemento que você já conhecia no Excel.

---

## 🗂️ Mapeamento de Abas do Excel

| Aba no Excel | Onde está na Plataforma | Caminho |
|--------------|------------------------|---------|
| **Objetivos Corporativos** | Dashboard + Objetivos | Menu lateral → "Dashboard" ou "Objetivos" |
| **OKRs Brasil (SXS/Hiter)** | Objetivos | Menu lateral → "Objetivos" (filtrar por unidade) |
| **Ações SXS** | Ações | Menu lateral → "Ações" |
| **Ações Hiter** | Ações | Menu lateral → "Ações" |
| **Acompanhamento Trimestral** | Dashboard | Indicador "Q1 2026" no topo do Dashboard |

---

## 📋 Onde encontrar cada informação

### 1. Objetivos e Key Results

| No Excel | Na Plataforma |
|----------|---------------|
| Categoria (Segurança, Pessoas, OpEx) | Objetivos → Pilar (indicador colorido à esquerda) |
| Objetivo do time | Objetivos → Título do card (ex: "Rentabilidade") |
| KR (Key Result) | Objetivos → Clique no objetivo → Lista expandida |
| Código do KR (1.1, 2.1, etc.) | Objetivos → Badge azul dentro do KR |
| Responsável do KR | Objetivos → Texto "Owner" abaixo do título |
| Baseline / Target / Atual | Objetivos → Texto abaixo do título do KR |
| Progresso (%) | Objetivos → Barra de progresso + Badge de % |

**Como editar um KR:**
1. Vá em "Objetivos"
2. Clique no objetivo para expandir
3. Passe o mouse sobre o KR
4. Clique no ícone de lápis (✏️)
5. Atualize o valor atual
6. Clique em "Salvar Alteração"

---

### 2. Ações (Iniciativas)

| No Excel | Na Plataforma |
|----------|---------------|
| KR relacionado | Ações → Badge "KR X.X" em cada ação |
| Iniciativa / Título | Ações → Título da ação |
| Detalhe | Ações → Descrição (visível ao criar/editar) |
| Responsável | Ações → Texto abaixo do título |
| Status | Ações → Badge colorido (Pendente, Em Progresso, Concluído, Bloqueado) |
| Data Limite | Ações → Ícone de calendário + data |
| KR Líder | Ações → Badge "KR X.X" |

**Como criar uma Ação:**
1. Vá em "Ações"
2. Clique em "+ Nova Ação"
3. Preencha: Título, Descrição, KR vinculado, Prioridade, Data Limite
4. Clique em "Criar Ação"

**Como mudar status:**
1. Passe o mouse sobre a ação
2. Clique nos 3 pontinhos (⋮)
3. Escolha: "Em Progresso", "Concluído" ou "Bloqueado"

---

### 3. Filtros por Unidade de Negócio

| No Excel | Na Plataforma |
|----------|---------------|
| Aba "SXS Brazil" | Dashboard/Objetivos → Dropdown "SXS Brazil" |
| Aba "Hiter" | Dashboard/Objetivos → Dropdown "Hiter" |
| Ver tudo junto | Dashboard/Objetivos → Dropdown "Todas Unidades" |

## 🎨 Legenda de Cores

### Progresso dos KRs
| Cor | Significado | Faixa |
|-----|-------------|-------|
| 🟢 Verde | No caminho certo | 70% - 100% |
| 🟡 Amarelo | Atenção necessária | 40% - 69% |
| 🔴 Vermelho | Crítico | 0% - 39% |

### Prioridade das Ações
| Cor | Significado |
|-----|-------------|
| 🔴 Vermelho | Alta prioridade |
| 🟡 Amarelo | Média prioridade |
| ⚪ Cinza | Baixa prioridade |

### Status das Ações
| Badge | Significado |
|-------|-------------|
| Pendente | Ainda não iniciado |
| Em Progresso | Trabalho em andamento |
| Concluído | Finalizado ✓ |
| Bloqueado | Impedido, precisa de ação |

## 🔍 Perguntas Frequentes

**P: Onde vejo o progresso geral?**
R: Dashboard → Card "Progresso Geral" com o círculo de porcentagem.

**P: Como sei quais KRs estão atrasados?**
R: Dashboard → Seção "Key Results que Precisam de Atenção" (ordenados do menor para maior progresso).

**P: Como filtro por SXS ou Hiter?**
R: Use o dropdown no topo direito das páginas Dashboard e Objetivos.

**P: Onde vejo as alterações feitas?**
R: Menu lateral → "Auditoria" para ver todo o histórico de mudanças.

**P: Os dados são salvos automaticamente?**
R: Sim! Todas as alterações são salvas no banco de dados imediatamente e registradas na auditoria.
